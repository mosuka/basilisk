Cluster APIs
============

Get Node API
------------

.. code-block:: text

    GET /-/_node


Add Node API
------------

.. code-block:: text

    GET /-/_node?node=<NODE_NAME>

* ``<NODE_NAME>``: The node name.


Delete Node API
---------------

.. code-block:: text

    DELETE /-/_node?node=<NODE_NAME>

* ``<NODE_NAME>``: The node name.
