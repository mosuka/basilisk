Cluster APIs
============

Get Node API
------------

.. code-block:: text

    GET /cluster/<NODE_NAME>

* ``<NODE_NAME>``: The node name.


Add Node API
------------

.. code-block:: text

    PUT /cluster/<NODE_NAME>

* ``<NODE_NAME>``: The node name.


Delete Node API
---------------

.. code-block:: text

    DELETE /cluster/<NODE_NAME>

* ``<NODE_NAME>``: The node name.
