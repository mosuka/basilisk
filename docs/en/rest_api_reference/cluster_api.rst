Cluster APIs
===========

Cluster API
-----------

.. code-block:: text

    GET /rest/_cluster


Add Node API
------------

.. code-block:: text

    GET /rest/_node?node=<NODE_NAME>

* ``<NODE_NAME>``: The node name.


Delete Node API
---------------

.. code-block:: text

    DELETE /rest/_node?node=<NODE_NAME>

* ``<NODE_NAME>``: The node name.
